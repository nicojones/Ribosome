Namespace Kernel

Classes summary
AppKernel

Front logic. Dispatches and manages the input flow thread.
Contains many of all the $hooks present in the framework.
Logger Logs messages to an external database. Configure it from /app/config/config.ini
Mailer Send an email using Mandrillapp. Set the <key> from /app/config/config.ini
Mailer_PHP Send emails from your own server with the library PHPMailer
Permission

Manages the permissions required for a specific call (Controller->action()).
If the user has not enough permissions, it returns a __throw_to action to redirect them to.
Router

This file manages the routing for incoming URLs. Returns a call (Controller->action()) for the specific route.
To use in conjunction with /src/config/routing.ini, where all the desired routes must be present.
